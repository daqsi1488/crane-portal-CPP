# Portal Crane Control System v2.0

## Описание

Система управления портальным краном с графическим интерфейсом для Windows. 
Программа позволяет управлять перемещением крюка, поворотом стрелы, контролировать вес груза, 
а также отслеживать энергопотребление и вести журнал событий с криптографической подписью.

## Функциональность

- **Управление краном**: подъём/опускание крюка (1 м), поворот стрелы (15°)
- **Установка груза**: 5, 10 или 20 тонн
- **Аварийная остановка и сброс**
- **Подключение/отключение крана** (эмуляция EtherCAT / Modbus TCP)
- **Просмотр технической информации**
- **Мониторинг энергопотребления** и рекомендации по оптимизации
- **Журнал событий** с криптографической подписью
- **Верификация подписей** для проверки целостности логов

## Требования

- Windows 7 / 8 / 10 / 11
- Нет дополнительных зависимостей (статическая сборка)

## Установка и запуск

1. Скачайте `crane.exe`
2. Запустите файл

## Управление

| Действие | Кнопка |
|----------|--------|
| Поднять крюк | RAISE (1m) |
| Опустить крюк | LOWER (1m) |
| Повернуть налево | LEFT (15 deg) |
| Повернуть направо | RIGHT (15 deg) |
| Установить груз 5т | 5 TONS |
| Установить груз 10т | 10 TONS |
| Установить груз 20т | 20 TONS |
| Аварийная остановка | EMERGENCY STOP |
| Сброс аварии | RESET |
| Подключение к крану | CONNECT CRANE |
| Отключение | DISCONNECT |
| Техническая информация | CRANE INFO |
| Энергопотребление | ENERGY OPT |
| Журнал событий | EVENT LOG |
| Проверка подписей | VERIFY SIGNATURE |
| Выход | EXIT |

## Логирование

Все события сохраняются в файл `crane_log.txt` в папке с программой. 
Каждая запись содержит временную метку, тип события, сообщение и криптографическую подпись....
